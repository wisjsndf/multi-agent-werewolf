# 狼人杀 7 人局核心规则手册

## 一、 阵营与胜负条件
本局游戏为 7 人局，包含：2名狼人，1名预言家，1名守卫，3名平民。
- **好人阵营（平民+神职）胜利条件**：放逐所有（2名）狼人。
- **狼人阵营胜利条件（屠边局）**：杀光所有（3名）平民，或者杀光所有（2名）神职。

## 二、 角色技能详解

### 1. 守卫 (Guard) - 神职
- **技能**：每晚可以暗中保护一名玩家（包括自己），如果该玩家当晚遭到狼人袭击，该玩家不会死亡（称为“平安夜”）。
- **绝对限制（同守限制）**：守卫【绝对不能】连续两晚守护同一名玩家。例如昨晚守了3号，今晚只能守除了3号以外的其他人，或者选择空守。

### 2. 预言家 (Seer) - 神职
- **技能**：每晚可以查验一名玩家的具体阵营。系统只会告知该玩家是“好人”还是“狼人”，但无法知晓具体的底牌（例如查验平民或守卫，结果都是“好人”）。
- **限制**：每晚只能查验一次。不能查验已经死亡的玩家。

### 3. 狼人 (Werewolf) - 狼人阵营
- **技能**：每天夜晚，狼人团队可以共同商议，并投票决定袭击一名玩家。得票最多的玩家将被杀害。
- **特权**：狼人在夜晚互相认识，知道彼此的座位号。

### 4. 平民 (Villager) - 好人阵营
- **技能**：没有任何夜晚技能，全程闭眼。白天通过听取发言和逻辑推理来寻找狼人。

## 三、 夜间结算优先级（极其重要）
当夜晚同时发生守卫技能和狼人袭击时，按以下逻辑结算：
- 若狼人刀了 A，且守卫守了 A：A 存活（被守卫救下）。
- 若狼人刀了 A，且守卫没有守 A：A 死亡。

## 四、 白天流程与规则
1. **死者宣布**：天亮后，上帝宣布昨晚死亡名单。
2. **遗言规则**：
   - 第一天夜晚死亡的玩家，有且仅有一次发表遗言的机会。
   - 第二天及以后夜晚死亡的玩家，没有遗言。
   - 白天被投票放逐的玩家，无论第几天，都有遗言。
3. **发言环节**：所有存活玩家轮流发言。
4. **投票放逐**：
   - 存活玩家投票选出心目中的狼人，得票最高者被放逐（死亡）。
   - **平局结算**：如果出现最高票数相同的平票情况，系统将**随机放逐**一名平票玩家。
   - 玩家可以自愿选择弃票。
